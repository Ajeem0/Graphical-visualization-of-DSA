# Graphical-visualization-of-DSA

#  DSA Playground App

This is a Python Tkinter project to visualize **Data Structure and Algorithms** concepts.

##  Features
- Sorting Visualizer
- Tree Visualizer
- Traversal (DFS, BFS)
- Heap Operations
- Sudoku Solver
